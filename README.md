# Social Media Factory

## Overview

Social Media Factory is an **n8n automation workflow** that streamlines the entire social media content lifecycle—from content creation to publishing and reporting.

The workflow uses **AI models (OpenAI & Gemini)** to generate platform-specific content, routes it through an approval process, creates or processes images, publishes posts across multiple social media platforms, and delivers a final report to stakeholders.

## Workflow Steps

### 1. Content Generation
- User submits a post idea or campaign details.
- AI generates content for:
  - LinkedIn
  - Instagram
  - Facebook
  - X (Twitter)

### 2. Approval Process
- Generated content is formatted into a review email.
- Content is sent for approval before publishing.
- Publishing proceeds only after approval.

### 3. Image Creation & Processing
- Generate AI images or use uploaded images.
- Resize and prepare images for different platforms.

### 4. Social Media Publishing
Automatically publishes approved content to:
- Instagram
- Facebook
- LinkedIn
- X (Twitter)

### 5. Reporting
- Collects publishing results.
- Generates a summary report.
- Sends notifications via Email and Telegram.
